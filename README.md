# MPCA-Hardware-Project
Automatic hand sanitizer dispenser with counter - detects a user’s hand using an IR sensor and dispenses sanitizer while maintaining a count of the number of uses.

## Project Description
This project is an automatic smart liquid dispenser system that detects objects using IR and ultrasonic sensors and dispenses liquid using a servo motor.

## Features
- Automatic object detection
- Distance checking
- Liquid level monitoring
- Low liquid alert
- Dispensing count tracking

## Components Used
- Arduino Uno
- Servo motor
- IR sensor
- Ultrasonic sensor
- Water level sensor
